# Invio di una mail con PHP

Inserire all'interno del file composer.json
la dipendenza con PHPMailer

```json
{
  "require" : {
    "phpmailer/phpmailer": "^6.8"
  }
}
```
e procedere all'aggiornamento con composer.

L'esempio semplice di utilizzo si trova in example.php

https://stackoverflow.com/questions/76186516/im-trying-to-make-phpmailer-work-but-it-keeps-giving-me-the-smtp-error-could-n
